# Murad Farzulla

Independent researcher. Founder & Director of **[Dissensus Ltd](https://dissensus.ai)**, which runs the Adversarial Systems & Complexity Research Initiative — **[ASCRI](https://systems.ac)**.

I work on adversarial systems: coordination friction, systemic risk, multi-agent systems, and AI governance. Methods are computational finance and formal verification (Lean 4). Each paper takes the same friction dynamics into a different domain — crypto market microstructure, political governance, AI alignment.
